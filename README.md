# CatApp
A simple app with cat breeds using https://thecatapi.com/ 

**Some of the libraries used for this app were:**

* Dagger2
* RxKotlin, RxJava2
* Android Architecture Components (LiveData, ViewModel, Databinding)
* Retrofit2
* MVVM Pattern
* Glide
